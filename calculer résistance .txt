/*Ecrire un algorithme qui affiche
 la résistance équivalente à trois
résistances R1, R2, R3 (type réel),
- si les résistances sont branchées en série:
Rsér = R1+R2+R3
- si les résistances sont branchées en parallèle:
Rpar = (R1*R2*R3)/(R1*R2+R1*R3+R2*R3)
*/
Algorithme résistance_eq
Variable
	R1, R2, R3, Rser, Rpar: réel
Début
	R1 <- 50
	R2 <- 100
	R3 <- 1000
	Rser <- R1 + R2 + R3
	Rpar <- (R1*R2*R3)/(R1*R2+R1*R3+R2*R3)
	ecrireln("Rsér = ", Rser, " ohm")
	ecrireln("Rpar = ", Rpar, " ohm")
Fin
